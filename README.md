# blog
A simple blog

------

**blog-app**：web前端文件，框架使用的是vue。目前完成前台展示的所有页面，组件包括分页组件、表单组件、时间线组件，对话框组件、3d云标签组件、评论列表组件等；

**blog-backend**：后端文件，框架使用的是python的flask框架，提供了前端展示调用的基本接口。
